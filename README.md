# ITPM Assignment 1 - Automation Testing

## 📌 Project Overview
This project evaluates the accuracy of converting Singlish (Sinhala typed using English letters) into Sinhala using automated testing with Playwright.

## 🎯 Objective
To identify incorrect transliterations and analyze system weaknesses using structured test cases.

## 🛠️ Technologies Used
- Python
- Playwright
- OpenPyXL

## 📂 Project Files
- test_automation.py → Main automation script
- Assignment 1 - Test cases.xlsx → Test cases and results
- Other supporting files

## ⚙️ Setup Instructions

### Step 1: Install Python
Download and install Python (version 3.11 or above)

### Step 2: Install Dependencies
Open Command Prompt and run:

pip install playwright openpyxl  
playwright install  

### Step 3: Navigate to Project Folder
Use Command Prompt:

cd D:\test_automation

## ▶️ Run the Automation Script

Run the following command:

python test_automation.py --excel "Assignment 1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator"

## 📊 Output
- Excel file will automatically update:
  - Actual Output
  - Status (Pass/Fail)

## 📌 Notes
- All test cases are designed to identify failures in Singlish to Sinhala conversion.
